# Ensemble Learning Pipeline with FastAPI Deployment

## 📌 Overview
This project demonstrates an ensemble learning pipeline using Logistic Regression, Random Forest, and Gradient Boosting, combined via stacking. The final model is deployed with FastAPI for real-time predictions.

## 📂 Structure
- `src/` → preprocessing, training, ensemble, evaluation scripts
- `models/` → saved ensemble model
- `api/` → FastAPI deployment
- `data/` → dataset (`students.csv`)
